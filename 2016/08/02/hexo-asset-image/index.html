在hexo中使用本地图片

使用hexo博客的时候，经常会插入本地博客，但是一般情况下都是把图片放在source/img下，在markdown里写![image](/source/img/imageName.png)。但是这样在本地的编辑器里不能正确识别图片的位置。

所以这里推荐一个插件，可以方便管理图片资源 asset-image。

使用

首先确认 _config.yml 中有 post_asset_folder: true。没有就写入一条。(冒号后面，空格，空格，空格。重要的事情要说三遍)

然后在 hexo 目录，执行

npm install https://github.com/CodeFalling/hexo-asset-image --save

之后创建文章的时候，仍然 hexo new ""就可以创建新的文章了，同时会在source/_post目录下自动生成同名的一个文件夹。

假设在

hexo-asset-image
├── 001.png
├── 002.png
└── 003.png    
hexo-asset-image.md

这样的目录结构（目录名和文章名一致），只要使用 ![logo](hexo-asset-image/001.png) 就可以插入图片。
