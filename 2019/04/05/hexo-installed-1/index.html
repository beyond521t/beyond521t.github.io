Mac+HEXO+Github 搭建属于自己的博客(一)

  • 前言
    • 准备工作
  • 博客初始化
  • 本地博客关联GitHub
  • 使用hexo写博客

1、前言

1、选择GitHub的原因不用多说，程序员的乐园，更是支持pages功能

2、免费方便，不用花一分钱就可以搭建一个自由的个人博客，不需要服务器不需要后台；

3、全是静态文件，访问速度快；

4、可以随意绑定自己的域名，不仔细看的话根本看不出来你的网站是基于github的；

5、数据绝对安全，基于github的版本管理，想恢复到哪个历史版本都行；

6、博客内容可以轻松打包、转移、发布到其它平台；

1.1 准备工作

1.1.1 github帐号

1.1.2 安装Node.js 和 Git

这里默认一般都是安装好的，可以命令行查看

node -v
v11.12.0

npm -v
6.7.0

git --version
git version 2.15.0

没有安装的，可以进行安装。Mac上安装可以选择图形化方式和终端安装，此处直接使用客户端方式安装. Node.js中文网下载文件（用来生成静态页面的），根据提示安装即可，安装成功后在目录 /usr/local/bin目录下。使用上述命令进行测试即可。

安装Git

作用：把本地的hexo内容提交到github上去.

安装了Xcode的自带有Git，我就不多说了。没有安装的建议使用Homebrew进行安装。

安装 Homebrew，终端执行

$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

然后执行如下命令安装 Git

sudo brew install git

1.1.3 安装Hexo

Node.js和Git都安装成功后开始安装Hexo。安装时注意权限问题，加上sudo，其中-g表示全局安装。

sudo npm install -g hexo

博客初始化

创建存储博客的文件夹，比如命名为myblog，然后进入到myblog目录中。

cd myblog

初始化本地博客，会自动下载一系列文件

hexo init

执行下述命令安装 npm

sudo npm install

执行下述命令生成本地网页文件并开启本地服务器，然后通过http://localhost:4000查看本地博客。

hexo g
hexo s

本地博客关联GitHub

3.1本地博客代码上传GitHub

注册并登陆GitHub账号后，新建仓库，名称必须为user.github.io，如beyond521t.github.io

终端cd到myblog文件夹下，打开_config.yml文件。或者用其他文本编辑器打开可以，推荐sublime。

打开后到文档最后部分，将deploy配置如下。
deploy:
type: git
repository: https://github.com/beyond521t/beyond521t.github.io.git
branch: master

其中将repository中beyond521t改为自己的用户名，注意type、repository、branch后均有空格。通过如下命令在myblog下生成静态文件并上传到服务器。

hexo g
hexo d

若执行hexo g出错则执行npm install hexo –save，若执行hexo d出错则执行npm install hexo-deployer-git –save。错误修正后再次执行hexo g和hexo d上传到服务器。

若未关联GitHub，执行hexo d时会提示输入GitHub账号用户名和密码，即:

username for 'https://github.com':
password for 'https://github.com':

hexo d执行成功后便可通过https://beyond521t.github.io访问博客，看到的内容和http://localhost:4000相同。

3.2 添加ssh keys到GitHub

添加ssh key后不需要每次更新博客再输入用户名和密码。首先检查本地是否包含ssh keys。如果存在则直接将ssh key添加到GitHub之中，否则新生成ssh key。
执行下述命令生成新的ssh key，将your_email@example.com改成自己注册的GitHub邮箱地址。默认会在~/.ssh/id_rsa.pub中生成id_rsa和id_rsa.pub文件。

ssh-keygen -t rsa -C "your_email@exampl"

Mac下利用cat ~/.ssh/id_rsa.pub 即可查看公钥信息，即为ssh key，将此信息复制到GitHub的Add ssh key路径GitHub->Setting->SSH keys->add SSH key中即可。Title里填写任意标题，将复制的内容粘贴到key中，点击Add key完成添加。

常用命令

hexo new"postName" #新建文章

hexo new page"pageName" #新建页面

hexo generate #生成静态页面至public目录

hexo server #开启预览访问端口（默认端口4000，'ctrl + c'关闭server）

hexo deploy #将.deploy目录部署到GitHub

hexo help # 查看帮助

hexo version #查看Hexo的版本

新版本的hexo，有简写命令，例如：

hexo generate --> hexo g
hexo server   --> hexo s
hexo deploy   --> hexo d
