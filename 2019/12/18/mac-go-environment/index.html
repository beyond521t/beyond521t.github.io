Mac 下Go语言开发环境设置

分别为 bash 和 zsh 两种命令方式

mac常用的shell是bash 和 zsh 两种命令方式

bash

如果你使用bash

一、修改文件~/.bash_profile 

vim ~/.bash_profile

二、在末尾添加自定义文件的位置

export GOPATH=${HOME}/Documents/golang 
export GOBIN=$GOPATH/bin
export PATH=$PATH:$GOBIN

三、

source ~/.bash_profile

四、使用go env查看是否修改

zsh

一、修改文件~/.zshrc 

vim ~./zshrc

二、在末尾添加自定义文件的位置

export GOPATH=${HOME}/Documents/golang 
export GOBIN=$GOPATH/bin
export PATH=$PATH:$GOBIN

三、

source ~/.zshrc

四、使用go env查看是否修改
